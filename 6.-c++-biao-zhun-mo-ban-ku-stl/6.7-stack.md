# 6.7 stack

```cpp
#include<stack>
using namespace std;
```

#### 6.7.1 stack的定义

`stack<typename> st;` 

#### 6.7.2 stack容器内元素的访问

只能通过`top()`函数来访问栈顶元素

#### 6.7.3 stack常用函数

1. `push(typename item)` ：O\(1\)
2. `top()` ：O\(1\)
3. `pop()` ：O\(1\)
4. `empty()` ：O\(1\)
5. `size()`：O\(1\)
